# 简易麻将

- **决定庄家**：随机决定庄家

## 🚀 使用方法

1. **修改配置文件（可选）**  
   可根据需要修改 `config` 文件夹中的内容，  
   具体说明详见：[config.md](./src/Config/config.md)

2. **运行后端服务器**  
   使用 Node.js 执行主程序：

   ```bash
   node main.js
   ```
3. **打开前端页面**  
   index.html
